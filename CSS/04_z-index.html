Z-index works on non static elements and it takes both -ve and +ve values.
Note: If elements that overlap do not have z-index specified then that element will show up that is mentioned last in document.

In normal flow blue box will appear above red
Using z-index to show red box above blue box
z-index: 1
z-index: 100
z-index: 2
z-index: 2
When two region overlap the parent/container z-index is used to determine which one comes on top
Here z-index=2 on overlap with blue box, but it comes on top since blue has z-index as 1
