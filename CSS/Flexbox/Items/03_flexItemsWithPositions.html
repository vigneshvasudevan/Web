Few points to remember
  1. An absolutely-positioned child of a flex container does not participate in flex layout
  2. The static position of an absolutely-positioned child of a flex container is determined such that the child is positioned as if it were the sole flex item in the flex container, assuming both the child and the flex container were fixed-size boxes of their used size.
